# crypto-price-cli

Crypto Price Command Line in Rust to fetch spot market from FTX

> just experiment cli that I learned to create my first cli app with Rust & Clap

## Usage

```bash
# default is BTC/USD
cargo run

# fetch ETH/USD
cargo run -- -m ETH/USD

# help
cargo run -- --help
```
